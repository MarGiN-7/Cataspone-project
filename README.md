## Overview

An end-to-end machine learning project covering customer churn prediction, credit risk analysis, and marketing campaign effectiveness. This project demonstrates a complete data science pipeline from exploratory analysis to model evaluation.

## Features

- 🎯 **Customer Churn Prediction** - Identify at-risk customers
- 💳 **Credit Risk Analysis** - Assess financial risk profiles
- 📊 **Marketing Campaign Analysis** - Evaluate campaign effectiveness
- 📈 **Comprehensive Visualization** - Interactive insights and reports

## Usage
Run the pipeline in sequence (churn → credit risk → marketing) using Python. Required libraries: [numpy, pandas, scikit-learn, matplotlib, seaborn]. Each section loads its dataset automatically from public URLs, processes data, trains models, and outputs insights + visualizations.

## Results

- Churn Prediction Accuracy: ~80% (Random Forest model)
- Credit Risk AUC: ~0.78 (Random Forest model)
- Campaign ROI Improvement: Best campaign acceptance ~15% higher than baseline
